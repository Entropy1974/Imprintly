Builder Tab — How It Works

Entry Point 1 — "Current Design": User has customized a card through stages 2-4 (Content, Media, Design) and wants to save their current design as a reusable template. Colors, fonts, ribbon, and element toggles are pre-filled from the active card.

Entry Point 2 — "From Scratch": Start fresh. Pick a base layout, set default colors/fonts/ribbon, choose which elements to include. Great for building a template library before filling in contact details.

Base Layout: Maps to existing BASE_TEMPLATES (Split, Centered, Minimal, Bold, Modern, Luxury). Determines overall card structure — where the logo, text, and ribbon go.

Default Colors: Primary, secondary, accent, and background. When "Current Design" is selected, these pull from the active card state. Colors include gradient options.

Preview: Live front + back thumbnails rendered using the same card pipeline — templates look like real cards, not generic placeholders.

Saved templates appear in the Gallery tab under a "My Templates" category. They can be applied, edited, or deleted.
